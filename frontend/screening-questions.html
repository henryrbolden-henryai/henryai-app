What are screening questions?

Some job applications include written questions before you submit. These are different from recruiter phone screens.

Where to find them:

  • Click "Apply" on the company's job posting
  • Look for required questions before the submit button
  • Common on Greenhouse, Lever, and Workday applications

Not all applications have them. If the application only asks for resume/cover letter, you can skip this section.
